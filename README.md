# 🍳 AI Kitchen Companion

An AI-powered cooking application designed primarily for dorm students — but built for anyone who wants smarter, stress-free cooking.

---

## 🚀 Overview

AI Kitchen Companion transforms everyday cooking into a guided, intelligent experience. Whether you're working with limited ingredients, short on time, or unsure how to fix a cooking mistake, the app provides real-time assistance powered by AI.

**Our mission:**  
Reduce food waste • Simplify cooking • Build confidence in the kitchen
